# v2RayTun Linux

Linux support is planned for v2RayTun.

## Availability

- Will be available on [GitHub Releases](https://github.com/LXST-CODE/v2RayTun/releases).
- Will be available on [Telegram Releases](https://t.me/v2raytun/3).

## Requirements

- Requires Ubuntu 20.04 LTS or higher
- Requires Debian 10 or higher
- Requires GTK 3 runtime libraries
- Supported CPU architecture: x64

## Notes

The Linux version may differ from other platforms in UI, feature availability, release schedule, packaging, and platform-specific behavior.

Some features, such as TUN mode or system-level network integration, may require additional permissions or system configuration.
