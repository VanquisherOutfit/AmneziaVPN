# v2RayTun Windows

v2RayTun is supported on Windows 10 and higher. It may work on lower versions down to Windows 8.1, but this is not guaranteed.

## Availability

- Available on [GitHub Releases](https://github.com/LXST-CODE/v2RayTun/releases).
- Available on [Telegram Releases](https://t.me/v2raytun/3).

## Requirements

- Minimum Windows version: Windows 10
- Supported CPU architecture: x64

## Notes

The Windows version may differ from other platforms in UI, feature availability, release schedule, and platform-specific behavior.
