# v2RayTun Android

v2RayTun is supported on Android 7 and higher. It may work on lower versions down to Android 5, but this is not guaranteed.

## Availability

- Available on [Google Play](https://play.google.com/store/apps/details?id=com.v2raytun.android).
- Available on [GitHub Releases](https://github.com/LXST-CODE/v2RayTun/releases).
- Available on [Telegram Releases](https://t.me/v2raytun/3).

## Requirements

- Minimum Android version: 7
- Minimum API level: 21
- Supported CPU architecture: x86, x86_64, armeabi-v7a, arm64-v8a

## Notes

The Android version may differ from other platforms in UI, feature availability, release schedule, and platform-specific behavior.

The Android TV version may differ from the regular Android version in TV-focused UI.
